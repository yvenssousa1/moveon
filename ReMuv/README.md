# ReMuv

Aplicativo premium de reabilitação e fisioterapia, inspirado no Freeletics, com lógica clínica adaptativa, onboarding imersivo, player de vídeo, gamificação, paywall e integração com assets remotos.

## Estrutura premium
- Onboarding detalhado
- Player de vídeo com feedback adaptativo
- Gamificação (streak, pontos)
- Paywall (planos e monetização)
- Lógica clínica adaptativa
- Integração com vídeos remotos (API/Storage)

## Tecnologias sugeridas
- React Native (expo)
- TypeScript
- Styled Components ou Tailwind
- Context API/Zustand
- Notificações Push

## Instruções
Todas as melhorias e novas features devem ser aplicadas nesta raiz (`ReMuv`).
